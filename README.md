# arithmetic-px
解析引擎

目的: 数据质量 配置解析sql
      
常见的几个解析

1.非空解析        --is null 检查
2.数值解析        --正则表达式匹配
3.取值范围解析    --字段级检查
4.自定义解析      --自定义sql检查
5.数据格式解析    -- 表结构检查
6.唯一性解析      -- 唯一性检查

功能目标
    1.支持单表单字段检查
    2.支持单表多字段检查
    3.支持解析sql
    4.支持检核指标
    5.支持检核记录
    6.支持全量匹配
    7.支持校验sql
    8.支持sql脚本预览
    9.支持freemark变量替换
    10.支持表达式二叉树解析  true and false or false and true / 1 + 1 -2 -3 + 4 
    11.支持二叉树反解析
    
模块分类
    1.sql解析器
    2.sql校验器
    3.sql执行器
    4.sql生成器
    5.sql反编译器
